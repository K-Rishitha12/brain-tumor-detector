About This Project

Project Overview

This system detects brain tumors using MRI images with high accuracy. It combines CNN and SVM for predictions. Users can upload MRI scans and get instant results.

Technologies Used

  • Python
  • Flask
  • HTML, CSS, JavaScript
  • CNN (Convolutional Neural Network)
  • SVM (Support Vector Machine)
  • SQLite
  • MRI Dataset

Objectives of the Project

  • Provide accurate brain tumor detection using MRI scans.
  • Help doctors make faster diagnosis decisions.
  • Enable a user-friendly interface for patients and medical staff.

Features of the System

  • Secure login system for users.
  • Upload MRI images for analysis.
  • Automatic tumor detection and prediction.
  • View results instantly with clear classification.
